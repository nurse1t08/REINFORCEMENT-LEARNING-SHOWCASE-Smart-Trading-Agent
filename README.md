# Smart Trading Agent

## Описание проекта

Данный проект представляет собой RL-симулятор торговли с использованием алгоритма Q-learning.

AI-агент перемещается по торговому центру, анализирует цены в магазинах, покупает выгодные товары и продаёт их на рынке для получения максимальной прибыли.

Проект включает обучение агента, визуализацию среды, отображение стратегии агента и интерактивный веб-интерфейс на Streamlit.

## Цель проекта

Цель проекта — продемонстрировать работу reinforcement learning на простой торговой среде, где агент обучается:

- искать самые выгодные магазины
- избегать убыточных покупок
- эффективно перемещаться по карте
- управлять инвентарём
- максимизировать прибыль
- минимизировать количество лишних действий

## Установка и запуск

### 1. Клонировать репозиторий

git clone https://github.com/nurse1t08/smart-trading-agent.git
cd smart-trading-agent

### 2. Создать виртуальное окружение

python3 -m venv .venv
source .venv/bin/activate

### 3. Установить зависимости

pip install -r requirements.txt

### 4. Обучить модель

python3 train.py

После выполнения будет создан файл:

- model.pkl

## Запуск веб-приложения

streamlit run app.py

## Открыть в браузере

http://localhost:8501

## Как работает модель

В проекте используется:

- Q-learning для обучения агента
- Q-table для хранения стратегии
- reward shaping для улучшения обучения
- система штрафов за неэффективные действия
- inventory management
- динамическая торговая среда

Агент получает rewards за:

- прибыльные покупки
- успешные продажи
- короткие маршруты
- эффективные действия

А также получает штрафы за:

- бесполезные перемещения
- повторение маршрутов
- невыгодные покупки
- преждевременную продажу
